Networking Archives

‘Networking Archives’ was a three-year, AHRC funded research project which merged the early modern correspondence data collected in ‘Early Modern Letters Online’ with metadata from the ‘Gale State Papers Online’ to create the UK’s largest meta-archive of curated early modern correspondence metadata.

This site archives the project’s tools, research, and resources. It includes a number of tools which can be used to analyse or browse the project’s correspondence data. You can find extensive teaching materials which can be freely re-used, including an interactive glossary of network terms.
